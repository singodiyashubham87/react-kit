## Getting Started
Follow these steps to get started with GitWrap:

### Prerequisite

- Node.js (https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
```bash
git clone https://github.com/singodiyashubham87/React-Kit.git
cd React-Kit
```
   
2. Install dependencies:
```bash
npm install
```
3. Start the app:
```bash
npm run dev
  ```
